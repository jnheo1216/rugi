## 개발환경 구성 (0830)

> ### :snake: Anaconda
>
> 설치: https://www.anaconda.com/products/individual
>
> 가성환경 생성 및 활성화
>
> ```
> conda create -n [name] python=3.7 # 가상환경 생성
> conda activate [name] # 가상환경 활성화
> conda install pytorch torchvision torchaudio -c pytorch # 프레임워크 및 라이브러리 설치
> ```
